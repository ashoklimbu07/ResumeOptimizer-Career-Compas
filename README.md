# Resume Optimizer

Resume Optimizer is a web application that helps users manage and improve resumes for better job applications. It combines resume storage, structured editing workflows, and AI-assisted analysis in one platform.

## What problem it solves

Job seekers often struggle with scattered resume versions, weak alignment with job descriptions, and missing optimization insights. Resume Optimizer centralizes resume management and provides data-driven feedback so users can prepare stronger, more targeted applications.

## What it includes

- Resume upload, preview, download, update, and deletion workflows.
- Persistent resume metadata and search with MongoDB.
- AI-powered resume parsing, improvement suggestions, and job-match scoring with missing keyword detection.
- User authentication and protected access so each user manages only their own resumes.
